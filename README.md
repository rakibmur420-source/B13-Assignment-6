# DigiTools

# Overview

DigiTools is a modern and responsive digital tools buying website developed using React.js.
The project allows users to browse different digital products, add products to cart, remove items, and proceed to checkout with a smooth and interactive user experience.

# Technologies Used
- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify
- JSON Data
# Features
- Fully responsive design (Mobile, Tablet, Desktop)
- Modern and clean user interface
- Dynamic product rendering from JSON data
- Add to cart functionality
- Remove product from cart
- Cart count update in Navbar
- Proceed to checkout functionality
- Toast notifications using React Toastify
- Product and Cart section toggling
- Interactive product cards
- Responsive layout with Tailwind CSS
- Smooth user experience
  # Sections Included
- Navbar
- Banner Section
- Stats Section
- Product Section
- Cart Section
- Footer
  
  # Cart Functionality
- Add products to cart
- Remove selected products
- Show total selected products
- Navbar cart count update
- Proceed to checkout clears all cart items
- Toast notifications for:
- Add to cart
- Remove product
- Checkout success
